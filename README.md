# Metodo-ZS-Buffer
Z-Buffer (GPU) vs S-Buffer (CPU) es una aplicación interactiva en Python que compara técnicas de ocultamiento de superficies. Incluye rasterización por software, shaders Phong y Gouraud, y visualización en tiempo real con PyQt5, PyOpenGL y NumPy, permitiendo analizar rendimiento y precisión entre CPU y GPU.
